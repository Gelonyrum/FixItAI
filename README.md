📸 **Vision & OCR:** Витягуйте текст із зображень у буфері обміну або заставте AI описати, що зображено на картинці.

💬 **Persistent Chat:** Повноцінний чат-інтерфейс (на базі tkinter), що підтримує історію повідомлень та контекст бесіди.

⌨️ **Гарячі клавіші (Hotkeys):**
-----------------------------------------------
`CapsLock` + `Num +`	  Виправлення граматики / Переклад на англійську

`CapsLock` + `Num -`	  Переклад на українську

`CapsLock` + `Num /`	  Вставити текст із шаблона

`CapsLock` + `Num *`	  Пояснити термін/текст/абревіатури

`CapsLock` + `Num 7`	  OCR: Витягти текст із картинки в буфері

`CapsLock` + `Num 4`	  OCR: Описати зображення з можливістю уточнити деталі в діалозі

`CapsLock` + `Num 6`	  Зробити Summary виділеного тексту

`CapsLock` + `Num 8/9`	Новий чат / Продовжити попередній

⚙️ **Налаштування**
-----------------------------------------------
• Відкрийте файл APIAndModel.txt у директорії програми.

• Додайте ваш Google AI API Key (можна згенерувати тут https://aistudio.google.com/api-keys), назву моделі (список моделей можна подивитись в контекстному меню в треї за допомогою пункту "List Available Models") та системний промпт (який буде застосовуватись як основа для спілкування в всіх вікнах де є можливість діалогу).

• Запустіть FixItAI.exe.

• Використовуйте Shift + CapsLock, щоб перемикати стандартний стан CapsLock.

🔨 **Збірка з вихідного коду (Build from Source)**
-----------------------------------------------
Якщо ви внесли зміни у код і хочете скомпілювати власний виконуваний файл, скористайтеся PyInstaller.

Попередня вимога:
Переконайтеся, що у вас встановлені всі залежності та папка Icons з іконкою проєкту знаходиться в кореневій директорії.

Консольна команда щоб зібрати exe файл:

`pyinstaller --noconfirm --onefile --windowed --name "FixItAI" --collect-submodules pynput --collect-submodules PIL --add-data "Icons;Icons" --icon="Icons/FixItAI.ico" FixItAI.py`

Логіка перехоплення клавіш винесена в окремий компонент для стабільності та швидкості роботи в Windows.

Зміна хоткеїв: Якщо ви хочете змінити комбінації клавіш (наприклад, замість CapsLock використовувати іншу клавішу), вам потрібно відредагувати вихідний файл .ahk у папці FixItAIHotkeysBlocker.

Перекомпіляція: Після внесення змін файл необхідно скомпілювати в .exe за допомогою Ahk2Exe (Знаходиться в папці Compiler).

Назва файлу: Готовий файл має називатися FixItAIHotkeysBlocker.exe і знаходитися у відповідній підпапці, щоб Python-скрипт зміг його ініціалізувати при запуску.
